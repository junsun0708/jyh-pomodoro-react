## Project overview (프로젝트 개요):
이 부분은 프로젝트의 전반적인 목적과 주요 특징을 설명합니다.

1. 이 프로젝트는 '포모도로 기법'을 기반으로 한 타이머를 통해 사용자의 생산성 향상을 돕습니다.
2. 포모도로 기법은 25분 작업과 5분 휴식을 반복하며, 4회 작업 후에는 30분간의 긴 휴식을 취합니다.
3. 미니멀한 디자인과 직관적인 UI로 사용자 경험을 향상시킵니다.

## Core functionalities (핵심 기능):
이 섹션에서는 포모도로 타이머의 주요 기능들을 나열합니다.

### 구현된 기능:
1. 타이머 기본 기능
   - 25분 작업 타이머
   - 5분 휴식 타이머
   - 4회 작업 후 30분 긴 휴식
   - 시작/일시정지 기능
   - 세션 카운트 표시
   
2. UI/UX 기능
   - 하늘색 그라데이션 배경
   - 원형 시계 디자인
   - 반응형 버튼 효과
   - 부드러운 애니메이션

### 구현 예정 기능:
1. 하단 네비게이션 기능
   - 명상모드
   - 타이머모드
   - 진행기록
   - 음악재생

2. 추가 기능
   - 알림 소리
   - 작업 내용 기록
   - 통계 기능
   - 설정 기능 (시간 조절 등)

## Tech Stack (기술 스택):
1. Frontend
   - Next.js 14.1.0
   - React 18
   - TypeScript
   - TailwindCSS
   - Material Icons

2. 주요 라이브러리
   - next/font: Geist 폰트 적용
   - Material Icons: UI 아이콘

## Design System:
1. 색상
   - 배경: 하늘색 그라데이션 (sky-400 to cyan-300)
   - 텍스트: 흰색
   - 버튼: 반투명 흰색 (white/20)

2. 타이포그래피
   - 기본 폰트: Geist Sans
   - 모노스페이스: Geist Mono
   - 아이콘: Material Icons

3. 컴포넌트
   - 타이머 디스플레이: 대형 숫자와 원형 점 디자인
   - 버튼: 반투명 효과와 호버 애니메이션
   - 네비게이션: 아이콘과 텍스트 조합

## 향후 개발 계획:
1. Phase 1 (현재)
   - 기본 타이머 기능 구현

2. Phase 2 (예정)
